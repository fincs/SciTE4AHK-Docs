SmartGUI Creator 4.0

This functionality is deprecated and will be removed in a future version of SciTE4AutoHotkey.

SmartGUI Creator is a WYSIWYG AutoHotkey GUI editor.

The build of SmartGUI Creator bundled with SciTE4AutoHotkey has some minor modifications:

  • The "Copy to Clipboard" functionality has been replaced with an "Insert into SciTE" functionality.
  • The final script generation routine has been edited.
  • The "Open" feature has been removed.
  • Unicode GUIs are supported.
