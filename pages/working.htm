Working with AutoHotkey scripts

Running AutoHotkey scripts

Running AutoHotkey scripts can be done with a simple keypress (F5) or by clicking the Run button in the toolbar.

Using the AutoHotkey Help

Looking up a certain command/function in the help file is very simple. In order to do so, select said command/function and press F1.

Compiling AutoHotkey scripts

Press Ctrl+F7 to compile your script or select Tools > Compile in the menu bar.

Selecting the AutoHotkey version

Functionality subject to change in a future version of SciTE4AutoHotkey.

You can choose which AutoHotkey executable you want to use for running and compiling scripts by clicking on the platform selection button in the toolbar (). This is useful if you need to check your scripts' compatibility with said builds.

Currently, SciTE4AutoHotkey looks for AutoHotkey v1.1 executables in the following locations:

  • ANSI (32-bit ANSI): %AhkDir%\AutoHotkeyA32.exe
  • Unicode (32-bit Unicode): %AhkDir%\AutoHotkeyU32.exe
  • x64 (64-bit Unicode): %AhkDir%\AutoHotkeyU64.exe

AutoHotkey v2 executables can also be used:

  • v2(x86): %AhkDir%\v2\AutoHotkey32.exe
  • v2(x64): %AhkDir%\v2\AutoHotkey64.exe
