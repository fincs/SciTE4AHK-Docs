What is it?

SciTE4AutoHotkey is a lightweight and easy to use SciTE-based AutoHotkey script editor. It provides:

  • Syntax highlighting
  • AutoComplete
  • AutoIndent
  • AutoHotkey help integration
  • Abbreviations
  • Editing macros
  • Interactive debugging
  • Powerful extension mechanisms
